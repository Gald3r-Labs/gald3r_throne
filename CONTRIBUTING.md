# Contributing to Gald3r Throne

> **⚠️ This is experimental, pre-1.0 software built by a solo developer.** Expect flaws.
> The single most helpful thing you can do is **report what breaks**.

Thank you for your interest in gald3r.

## Reporting Bugs

Open a [bug report](https://github.com/Gald3r-Labs/gald3r_throne/issues/new/choose) on this repo. Please include:

- Your OS and version (Windows / Linux)
- The Throne release version you downloaded
- Steps to reproduce, what you expected, and what actually happened
- Any error output or screenshots

For **security vulnerabilities**, do **not** open a public issue — email the maintainer directly.

## Requesting Features

Open a [feature request](https://github.com/Gald3r-Labs/gald3r_throne/issues/new/choose) describing the problem you're trying to solve, how you work around it today, and the ideal experience.

## Code Contributions

Gald3r Throne is developed by a **solo developer** and distributed as **compiled installers
only — it is not open source** (the application source is private). **Direct code contributions
(pull requests) are not accepted.**

All bug reports and feature requests are reviewed and may be incorporated into gald3r's
development. Adopted ideas are implemented by the maintainer and credited in the release notes.

You are welcome and encouraged to:

- Build plugins, extensions, or other works that interoperate with gald3r (the license permits this)
- Report bugs and suggest improvements via Issues

## How gald3r Is Developed

gald3r is built with gald3r — its own task management, skill system, and quality gates develop it.
This repository hosts the **published installer downloads only**; the source lives in a private
monorepo.
